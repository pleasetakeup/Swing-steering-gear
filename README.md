##about Swing-steering-gear

    一个简单的功能的实现，基于arduino，实现通过按钮对舵机的控制。
##function
      
     1. 外接按键，有消除抖动功能，按键可计数
     2. 每按一次按键，舵机转动10度，完成摆动功能。0-10-20-30---90---170-180-170-160---90---0
     3. 随着舵机转动角度的变化，板载LED灯以不同的频率闪烁
##external conditions

     arduino IDE 1.8及以上版本
